JARS README

This directory contains several plugins you might want to try.
The are installed by default on Unix and Windows.

Acu - converts french accents to HTML &...; entities and vice versa
	(contributed by Romain Guy <RomainGuy@aol.com>)
AltGrPatch - enables the use of the AltGr key in Win32 JDK 1.1
	(contributed by Juha Lindfors <jplindfo@cc.helsinki.fi>)
GenerateText - the one you've all been waiting for, a random text
	generator
Reverse - reverses the selection
Rot13 - everybody's favourite cypher
