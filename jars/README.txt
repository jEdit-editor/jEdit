JARS README

This directory contains several simple plugins for jEdit.
They should give you an idea of how jEdit plugins are written.

Plugins that are actually useful can be downloaded from
jEdit plugin central <http://www.icg-online.de/jedit/>.

GenerateText - the one you've all been waiting for, a random text generator
Reverse - reverses the selection
Rot13 - everybody's favourite cypher
