Introduction

 With jEdit's OptionPane API, new tabs can be added to the `Global Options' dialog.

The OptionPane Class

 The OptionPane is what all option pane classes must extend. It is a subclass of javax.swing.JPanel.

Registering OptionPanes

 To register an OptionPane class with jEdit, the class instance must be passed to the addOptionPane() method of the org.gjt.sp.jedit.jEdit class This should be done in the start() method of the plugin - see the jEdit Plugin Development help topic for details.
