Introduction

 With jEdit's Mode API, new edit modes can be written.

The Mode Class

 The org.gjt.sp.jedit.Mode class is what all edit modes must extend.

Registering Modes

 New edit modes can be added to jEdit's mode list with te addMode() method of the org.gjt.sp.jedit.jEdit class.

Modes should be registered in the start() method of the plugin - see the jEdit Plugin Development help topic for details.
