Introduction

 To write a jEdit command, an implementation of the org.gjt.sp.jedit.EditAction class must be provided, and an instance of the command must be added to jEdit's command list with either the addAction() or addPluginAction() method of the org.gjt.sp.jedit.jEdit class.

The difference between the two methods is that addAction doesn't list the action in the `Plugins' menu, while addPluginAction() does. addPluginAction() should be used in most circumstances.
