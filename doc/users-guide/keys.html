Introduction

 Within jEdit, keyboard shortcuts are written as modifier+key, where modifier is S for Shift, C for Control, or A for Alt. key is either a letter, or a named key such as PageUp.

Some keyboard shortcuts consist of more than one keystroke, for example `Delete Paragraph' is C+d C+e. Both key strokes must be pressed in the respective order.

Menu Item Shortcuts

The most frequently used menu item commands have keyboard shortcuts. They are listed to the right of the menu item's label.

Menu Mnemonics

 Pressing the Alt key in combination with one of the underlined characters in the menu bar activates the respective menu. Once a menu is open, an item in it can be selected by pressing the underlined character in its label.

While a menu is open, the arrow keys can be used to select menu items, and Space or Enter will activate the selected menu item.

The F10 key can be used to activate the menu bar for menu selection via arrow keys.

Dialog Shortcuts

All dialogs have an OK button which can be activated by pressing the Enter key.

Most dialogs also have a Cancel button which can be activated by pressing the Escape key.

Tab and Shift+Tab can be used to shift the keyboard focus to the next and previous component in a dialog box.

Text Editing Shortcuts

The following shortcuts are available in most text components, including the main editor component and the text fields in various dialogs.

If Shift is held down in combination with any of these keys, the selection is extended, rather than the caret being moved.
Backspace Deletes the character before the caret. If there is a selection, it will be deleted instead.
Delete Deletes the character after the caret. If there is a selection, it will be deleted instead.
Left, Right, Up, Down Moves the caret in the respective direction.
C+Left, C+Right Moves the caret one word backward and one word forward, respectively.
Home Moves the caret to the start of the line.
End Moves the caret to the end of the line.
C+Home Moves the caret to the start of the text.
C+End Moves the caret to the end of the text.
PageUp Moves the caret up one screenful of text.
PageDown Moves the caret down one screenful of text.

Customizing Keyboard Shortcuts

Menu item shortcuts can be customized in `Tools->Global Options->Shortcuts'.

Some notes about the key binding editor:

  • The key bindings are specified as described in the first section of this file - modifiers+key
  • The modifiers is optional - so for example you can bind to function keys, etc
  • Each key binding can consist of up to two key strokes
  • To reset to the default keys, you must delete your .jedit-props file, or if you don't want to lose your settings, open it with a text editor and remove all xxx.shortcut entires
