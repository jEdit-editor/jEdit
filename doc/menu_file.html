File Menu

Command Shortcut Description
New File C+n Creates a new `Untitled' file.
New View C+e C+n Creates a new view.
Open File C+o Opens the file specified in the `Open File' dialog.
Open URL C+e C+o Opens the file or URL entered in the `Open URL' dialog.
Open Selection Opens the selected file or URL.
Open Recent A menu of recently opened files. The number of recent files can be specified in the Options dialog box.
Close File C+w Closes the current file. If the file contains unsaved changes, you will be prompted if they should be saved.
Close View C+w Closes the current view.
Save C+s Saves the current file to disk. If the file is new, you will be prompted to enter a name first.
Save As Saves the current file to disk, after prompting for a new file name.
Save to URL Saves the current file to disk, after prompting for a URL or file name.
Send Displays a dialog for sending the current file via e-mail
Print C+p Displays a dialog for printing the current file.
Exit C+q Exits jEdit, prompting if any unsaved changes should be saved first.
