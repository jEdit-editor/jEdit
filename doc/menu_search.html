Search Menu

Note: To make full use of the Search & Replace features, see the Regular Expressions help topic.
Command Shortcut Description
Find C+f Displays the search and replace dialog.
Find Next C+g Locates the next occurance of the search string in the current document.
Find Selection C+e C+s Locates the next occurance of the selected string in the current document.
Replace C+e C+r Replaces the currently selected string with the replacement string.
Replace & Find Next C+e C+g Replaces the currently selected string with the replacement string, and locates the next occurance of the search string.
Replace All C+e C+a Replaces all occurances of the search string with the replacement string.
HyperSearch C+h Displays the HyperSearch dialog, which lists all lines the current document that contain a specified string.
Go to Line C+l Moves the caret to the start of the line specified in the dialog.
Set Marker C+e C+m Adds a marker at the caret position after prompting for a name. Markers are saved across editing sessions.
Clear Marker A submenu of markers. Selecting one will remove it from the document.
Go to Marker A submenu of markers. Selecting one will go to it.
