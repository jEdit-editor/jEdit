Bracket Highlighting

 If the character before the caret is a bracket, the matching bracket is now highlighted.

Current Line Highlighting

 The line the caret is on now has a different background color.

Syntax Colorizing Updates

 The syntax colorizing subsystem is a lot smoother in jEdit 1.3. Major changes from 1.2 are:
  • JavaScript mode added, for editing JavaScript scripts
  • The HTML mode now colorizes JavaScript inside <script> tags
  • C, C++ and Java colorizing is a little bit faster because of several optimizations
Many syntax colorizing related bugs were also fixed; see Bug Fixes for details.

Options Dialog Box Updates

 The `Options' dialog box has been completely overhauled. It can now change many more settings, including colors.

Desktop Saving

 When jEdit exits, a list of open files and caret positions is saved to a `desktop' file. When jEdit is started with no command line arguments, it reads this `desktop' file and opens the file(s).

New Commands

 The following text editing commands are new in jEdit 1.3:
  • Commands for manipulating lines
  • Commands for manipulating paragraphs
  • Commands for shifting the indent
  • Commands for manipulating the anchor, a floating marker that is useful in many situations
  • Commands for converting tabs to spaces and vice versa
  • `Paste Previous' command for pasting previously copied strings - by default, the last 100 strings are remebered
  • `Filter Selection' command that writes the selected text to the command's standard input, replacing the selection with the output
  • `Make' command that invokes the `make' tool, usually `make' on Unix and `nmake' on Windows
  • `Compile' command that invokes a mode-specific compiler

New Key Bindings

 The key bindings for a lot of commands have changed. jEdit now supports multiple keystroke key bindings.

Bug Fixes

 jEdit 1.3 fixes the following bugs present in 1.2:
  • Several Java and C keywords were missing
  • jEdit.regionMatches() had a bug, among other things this caused problems in HTML mode
  • Makefile mode didn't colorize short variables properly
  • A \$ inside a $...$ construct wasn't colorized correctly in TeX mode
  • Path names like ../../file could cause problems
  • More than one copy of a URL could be open at once
  • A failed `Save As' could cause weird problems
  • Search and replace didn't support ^ and $
  • `Replace All' command didn't work sometimes
  • Various Format command bugs
  • Various Expand Abbreviation command bugs
  • Send command didn't set the subject line properly
  • print.java didn't compile with Sun's javac
  • Auto indent didn't work properly with tab sizes other than 8
  • Removed debug message from `Spaces to Tabs' command
