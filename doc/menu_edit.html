Edit Menu

Command Shortcut Description
Undo C+z Undoes the previous editing operation.
Redo C+r Undoes a previous undo.
Cut C+x Places the selected text in the system clipboard, removing it from the document.
Copy C+c Places the selected text in the system clipboard, leaving it in the document.
Paste C+v Inserts the system clipboard contents at the caret position, replacing the selection, if any.
Paste Previous C+e C+v Displays a dialog with a list of previously copied text strings. The number of strings to remember can be specified in the Options dialog.
Clear Removes the selection from the document.
Select All C+a Selects the whole document.

Anchor Operations Submenu

Command Shortcut Description
Set Anchor C+t Moves the anchor to the caret position.
Go to Anchor C+y Moves the caret to the anchor position.
Select to Anchor C+u Extends the selection to the anchor position.
Exchange Anchor C+e C+x Exchanges the anchor and the caret position. Once the anchor is set, can be used to quickly jump between two locations.

Line Operations Submenu

Command Shortcut Description
Go to End of Indent C+i Positions the caret on the first non-whitespace character of the current line.
Select Line No Indent C+e C+i Selects the current line, ignoring any leading white space.
Delete Line C+d C+d Deletes the current line.
Delete Line No Indent C+d C+i Deletes the current line, ignoring any leading white space.
Delete to Start of Line C+d C+s Deletes from the start of the current line to thecaret position.
Delete to End of Line C+d C+f Deletes from the caret position the end of the current line.
Join Lines C+j Joins the current and the next line, deleting any white space from the start of the next line.
Scroll to Current Line C+e C+l Scrolls to the current line, positioning it at the center of the viewport.

Paragraph Operations Submenu

Command Shortcut Description
Go to Next Paragraph C+DOWN Moves the caret to the start of the next paragraph. A paragraph is a block of text ending with a double newline.
Go to Previous Paragraph C+UP Moves the caret to the start of the previous paragraph. A paragraph is a block of text ending with a double newline.
Select Next Paragraph CS+DOWN Extends the selection to the start of the next paragraph. A paragraph is a block of text ending with a double newline.
Select Previous Paragraph CS+UP Extends the selection to the start of the previous paragraph. A paragraph is a block of text ending with a double newline.
Delete Paragraph C+d C+p Deletes the paragraph containing the caret. A paragraph is a block of text ending with a double newline.

Code Block Operations Submenu

Command Shortcut Description
Select Code Block C+[ Selects the bracket enclosed block containing the caret. Brackets are often used to delimit code blocks in programming languages.
Shift Indent Left A+LEFT Removes one level of indentation from the start of every line in the selection, or the current line if the selection is empty. Tabs will not be used if the `noTabs' buffer-local property is set - see the Buffer-Local Properties help topic for details.
Shift Indent Right A+RIGHT Adds one level of indentation to the start of every line in the selection, or the current line if the selection is empty. Tabs will not be used if the `noTabs' buffer-local property is set - see the Buffer-Local Properties help topic for details.
Wing Comment C+e C+c Comments out the selected text, using mode-specific comment characters. For example, C code, when wing commented, looks like this: 
/* printf("%s: %s returned %d",argv[0],&cmdline,retval);
return 1; */
Box Comment C+e C+b Comments out the selected text, using mode-specific comment characters. For example, C code, when box commented, looks like this: 
/* if(retval == 0) regurtate_gunkulator(&params);
 * printf("%s: %s returned %d",argv[0],&cmdline,retval);
 * return 1; */
Block Comment C+e C+k Comments out the selected text, using mode-specific comment characters. For example, C code, when block commented, looks this like: 
// if(retval == 0) regurtate_gunkulator(&params);
// printf("%s: %s returned %d",argv[0],&cmdline,retval);
// return 1;
Command Shortcut Description
To Uppercase Converts all characters in the selection to upper case.
To Lowercase Converts all characters in the selection to lower case.
Format C+e C+f Formats the selection (or the whole document if the selection is empty) by removing duplicate spaces and splitting long lines. The maximum line length is specified in the `maxLineLen' buffer property - see the Buffer-Local Properties help topic for details.
Spaces to Tabs C+e C+t Converts consecutive spaces in the selection, or the current line if the selection is empty, to tab characters.
Tabs to Spaces C+e C+u Converts tab characters in the selection, or the current line if the selection is empty, to spaces.
Insert Date Inserts the current date and time at the caret position.
Expand Abbreviation C+b Attempts to complete the word before the caret by scanning backwards in the document and inserting the first string that matches the word. All non-alphanumeric characters except those listed in the `noWordSep' buffer property are treated as word separators - see the Buffer-Local Properties help topic for details.
Locate Bracket C+] Locates the bracket matching the one before the caret and moves the caret to it. The list of opening brackets is specified in the `openBrackets' property, the list of closing brackets in `closeBrackets'. See the Buffer-Local Properties help topic for details.
