Tools Menu

Command Shortcut Description
Execute Command Prompts for an operating system command to execute. Any output from the command is displayed in a `Command Output' window.
Filter Selection Prompts for an operating system command to execute. The command's input is the selected text, and any output replaces the selection.
Open Selection in WWW Browser Opens the selected URL in the WWW browser specified in the `Options' dialog box.
Open in WWW Browser Opens the current file in the WWW browser specified in the `Options' dialog box.
Compile C+k Invokes a mode-specific compiler to compile the current file. The compiler can also be specified in the `compiler' buffer property - see the Buffers help topic for details.
Make C+m Invokes the `make' tool to compile the current project. The make tool can be specified in the Options dialog or in the `make' buffer property - see the Buffers help topic for details.
Word Count Counts the characters words and lines in the current file.
Options Displays a dialog box for changing various settings.
Mode A submenu of available edit modes. See the Edit Modes help topic for details.
Plugins A submenu of installed plugins. See the Plugins help topic for details.
