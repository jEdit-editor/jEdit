Version 1.4pre1

New Features

  • Semi-intelligent indent in C/C++/Java modes
  • `Electric borders' - scrolling starts when caret is 4 lines from the top or bottom
  • `Generate Random Text' command added (it's only 2 Kb, it's funny, and it's useful in certain situations too)
  • Usage hints (ie, Tips of the Day) are displayed in the title bar until you start typing
  • Scroll to Line command added
  • Exchange Anchor and Caret command added
  • Go to End of Indent command added
  • Several commands for commenting out program source added

Enhancements

  • Status bar removed, dirty and read only status now displayed in the title bar, line number is displayed in the menu bar
  • Warning if file has changed on disk by another program
  • Keyboard shortcuts added changed and improved
  • Added a `Close' button to the help viewer
  • New menu item class displays key bindings in the correct place
  • Tab size setting on Options dialog is now an editable combo box
  • MacOS look and feel option added to Options
  • Auto save is now safer (first we save to #filename#+tmp+# then move to #filename# so that even if jEdit crashes during the autosave, the data will be preserved)
  • Auto save is now a little faster (flag is set if file has changed since previous auto save, and it is only saved then)
  • Auto save now occurs every 15 seconds rather than 30
  • Server messages are now prefixed with jEdit: so they are easier to track down
  • Command Output window is now an independent frame
  • Backup is only made only once per file visit (like Emacs)

Bug Fixes

  • Buffers menu's modified flag wasn't updated after a marker was set
  • Compiler setting wasn't saved
  • `Next Paragraph' and `Delete Paragraph' had problems deleting the last paragraph of a file
  • `Delete Line' had problems deleting the last line of a file
  • Pressing Escape in the `unsaved changes' dialog would discard changes, rather than cancelling the operation
  • Various dialogs didn't respond properly to Escape
  • Fixed incorrect message in `save first' dialog displayed by Compile command
